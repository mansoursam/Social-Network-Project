# Social-Network-Project

A simple Application built with "MERN Stack" to get connected with friends.

## Quick Start
npm install 
and then:
npm run dev
##Tools
React,Redux,MongoDB,expressJs,nodeJS...

Please make sure to update tests as appropriate.
